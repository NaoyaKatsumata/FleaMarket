<h1>概要</h1>

<h1>githubリンク</h1>
<p>https://github.com/NaoyaKatsumata/fleaMarket</p>
<h1>機能</h1>
<ul>
    <li>ユーザ会員登録</li>
    <li>ユーザログイン</li>
    <li>ログアウト</li>
    <li>商品一覧表示</li>
    <li>商品詳細取得</li>
    <li>商品お気に入り一覧表示</li>
    <li>ユーザ情報表示</li>
    <li>ユーザ購入商品一覧表示</li>
    <li>ユーザ出品商品一覧表示</li>
    <li>プロフィール変更</li>
    <li>商品お気に入り追加</li>
    <li>商品お気に入り削除</li>
    <li>商品コメント追加</li>
    <li>出品</li>
    <li>商品購入</li>
    <li>管理者登録</li>
    <li>管理者ログイン</li>
    <li>ユーザ削除</li>
    <li>管理者からのメール送信</li>
    <li>商品コメント削除</li>
</ul>

</ul>
